# 📖 User Manual / សៀវភៅណែនាំប្រើប្រាស់

## English Version

### For Staff Members

#### 1. Getting Started
1. Open Telegram
2. Search for your company's bot
3. Send `/start`
4. You'll see a welcome message with buttons

#### 2. Select Your Branch
**First time setup:**
- Click "🏪 Select Branch" button
- OR send `/branch`
- Choose your branch from the list
- You only need to do this once!

#### 3. Recording Payments

**Method 1: Quick Text Entry**
1. Click "📸 Record Payment"
2. Type: `AMOUNT DESCRIPTION`
3. Examples:
   - `25000 Fried rice`
   - `15000 Noodle soup`
   - `50000 Mixed order`
4. Click ✅ Confirm

**Method 2: With Screenshot**
1. Take a photo of KHQR payment
2. Send the photo to the bot
3. Type the amount and description
4. Click ✅ Confirm

#### 4. View Daily Summary
- Click "📅 Daily Summary"
- OR send `/daily`
- See total sales for today

#### 5. Common Commands
- `/start` - Show main menu
- `/branch` - Change branch
- `/record` - Record new payment
- `/daily` - Today's summary
- `/help` - Show help

### For Owners/Managers

#### 1. View Reports
Send `/report` and choose:
- 📅 Today - Today's sales
- 📆 This Week - Weekly summary
- 📊 This Month - Monthly summary
- 🏪 By Branch - Compare branches

#### 2. Daily Reconciliation
Send `/daily` to see:
- Total sales amount
- Number of transactions
- Sales by each branch
- Transaction count per branch

#### 3. Tips for Owners
- Check `/daily` every evening
- Compare branches weekly
- Backup data folder regularly
- Train staff on proper usage

## ខ្មែរ (Khmer Version)

### សម្រាប់បុគ្គលិក

#### 1. ការចាប់ផ្តើម
1. បើក Telegram
2. ស្វែងរកឈ្មោះ bot របស់ក្រុមហ៊ុន
3. ផ្ញើ `/start`
4. អ្នកនឹងឃើញសារស្វាគមន៍

#### 2. ជ្រើសរើសសាខា
**ដំបូង:**
- ចុច "🏪 Select Branch"
- ឬផ្ញើ `/branch`
- ជ្រើសរើសសាខារបស់អ្នក
- ធ្វើម្តងគត់!

#### 3. កត់ត្រាការទូទាត់

**វិធីទី១: បញ្ចូលលេខ**
1. ចុច "📸 Record Payment"
2. វាយ: `ចំនួនទឹកប្រាក់ ពិពណ៌នា`
3. ឧទាហរណ៍:
   - `25000 បាយឆា`
   - `15000 គុយទាវ`
   - `50000 ចម្រុះ`
4. ចុច ✅ យល់ព្រម

**វិធីទី២: ជាមួយរូបភាព**
1. ថតរូប KHQR payment
2. ផ្ញើរូបភាពទៅ bot
3. វាយចំនួនទឹកប្រាក់និងពិពណ៌នា
4. ចុច ✅ យល់ព្រម

#### 4. មើលសង្ខេបប្រចាំថ្ងៃ
- ចុច "📅 Daily Summary"
- ឬផ្ញើ `/daily`
- មើលការលក់សរុបថ្ងៃនេះ

#### 5. ពាក្យបញ្ជាសំខាន់ៗ
- `/start` - បង្ហាញម៉ឺនុយ
- `/branch` - ប្តូរសាខា
- `/record` - កត់ត្រាការទូទាត់ថ្មី
- `/daily` - សង្ខេបថ្ងៃនេះ
- `/help` - ជំនួយ

### សម្រាប់ម្ចាស់/អ្នកគ្រប់គ្រង

#### 1. មើលរបាយការណ៍
ផ្ញើ `/report` ហើយជ្រើសរើស:
- 📅 ថ្ងៃនេះ - ការលក់ថ្ងៃនេះ
- 📆 សប្តាហ៍នេះ - សង្ខេបប្រចាំសប្តាហ៍
- 📊 ខែនេះ - សង្ខេបប្រចាំខែ
- 🏪 តាមសាខា - ប្រៀបធៀបសាខា

#### 2. ការផ្សះផ្សាប្រចាំថ្ងៃ
ផ្ញើ `/daily` ដើម្បីមើល:
- ចំនួនទឹកប្រាក់សរុប
- ចំនួនប្រតិបត្តិការ
- ការលក់តាមសាខា
- ចំនួនប្រតិបត្តិការក្នុងមួយសាខា

#### 3. ដំបូន្មានសម្រាប់ម្ចាស់
- ពិនិត្យ `/daily` រៀងរាល់ល្ងាច
- ប្រៀបធៀបសាខារៀងរាល់សប្តាហ៍
- ធ្វើបម្រុងទុកទិន្នន័យជាប្រចាំ
- បណ្តុះបណ្តាលបុគ្គលិកឱ្យប្រើប្រាស់ត្រឹមត្រូវ

## Common Questions / សំណួរញឹកញាប់

### Q: Can I change my branch?
**A:** Yes! Send `/branch` anytime to select a different branch.

### Q: What if I make a mistake?
**A:** Contact your manager. They can check the transaction history.

### Q: Do I need internet?
**A:** Yes, you need internet connection to use the bot.

### Q: Is my data safe?
**A:** Yes, all data is stored securely and only accessible by authorized users.

### Q: តើខ្ញុំអាចប្តូរសាខាបានទេ?
**ចម្លើយ:** បាន! ផ្ញើ `/branch` គ្រប់ពេលដើម្បីជ្រើសរើសសាខាផ្សេង។

### Q: ប្រសិនបើខ្ញុំធ្វើខុស?
**ចម្លើយ:** ទាក់ទងអ្នកគ្រប់គ្រង។ គាត់អាចពិនិត្យប្រវត្តិប្រតិបត្តិការ។

### Q: តើខ្ញុំត្រូវការអ៊ីនធឺណិតទេ?
**ចម្លើយ:** បាទ/ចាស អ្នកត្រូវការអ៊ីនធឺណិតដើម្បីប្រើ bot។

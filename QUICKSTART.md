# 🚀 Quick Start Guide

Get your Bakong Vendor Bot running in 5 minutes!

## Prerequisites
- ✅ Windows computer
- ✅ Internet connection
- ✅ Telegram account

## Installation (5 Steps)

### Step 1: Install Node.js (2 minutes)
1. Download: https://nodejs.org/
2. Run installer
3. Click "Next" until done

### Step 2: Get the Code (1 minute)
1. Download this project as ZIP
2. Extract to `D:\Projects\DUC2026`

### Step 3: Run Installation (1 minute)
1. Double-click `install.bat`
2. Wait for installation to complete

### Step 4: Create Telegram Bot (2 minutes)
1. Open Telegram
2. Search: `@BotFather`
3. Send: `/newbot`
4. Choose name: `My Vendor Bot`
5. Choose username: `myvendor_bot`
6. **Copy the token!**

### Step 5: Configure & Run (1 minute)
1. Open `.env` file (created by install.bat)
2. Paste your token:
   ```
   TELEGRAM_BOT_TOKEN=paste_your_token_here
   ```
3. Save the file
4. Run: `npm start`

## ✅ Done!

Your bot is now running! 

### Test it:
1. Open Telegram
2. Search for your bot
3. Send `/start`
4. Follow the instructions

## 📱 How to Use

### For Staff:
1. `/start` - Start the bot
2. `/branch` - Select your branch
3. Send payment: `25000 Fried rice`
4. Confirm ✅

### For Owners:
1. `/daily` - See today's sales
2. `/report` - Detailed reports
3. All data saved automatically!

## 🆘 Need Help?

**Bot doesn't respond?**
- Check if `npm start` is running
- Make sure you used the correct bot username
- Try `/start` command

**Installation failed?**
- Make sure Node.js is installed
- Run `npm install` again
- Check internet connection

**Can't find .env file?**
- Run `install.bat` again
- Or copy `.env.example` to `.env` manually
